# Weekly Workflow SOP

## Monday
- Build niche list + city cluster
- Screen 20+ prospects

## Tuesday-Thursday
- Record/send Loom audits
- Run full follow-up cadence
- Book calls quickly

## Friday
- KPI review
- Pipeline cleanup
- Script/message improvements based on replies
- Plan next week niche focus

## Weekly Review Questions
- Which niche replied most?
- Which opening line got responses?
- Where are leads dropping in pipeline?
- Which objections repeated?
