# Rapid Audit Checklist (5-Minute Screen)

Use for triage before deciding to record Loom.

## Pass / Fail Gate (record only if 2+ fail)

- [ ] 5-second clarity: who/what/where is obvious
- [ ] Primary CTA visible above fold
- [ ] Mobile readability acceptable
- [ ] Trust cues visible early
- [ ] Contact path friction is low
- [ ] Site looks current and credible

## Opportunity Flags (mark all)

- [ ] Generic or weak headline
- [ ] No clear service area mention
- [ ] CTA hidden/vague/inconsistent
- [ ] Sparse or weak trust proof
- [ ] Poor section hierarchy
- [ ] Thin service pages
- [ ] Mobile layout issues
- [ ] Obvious speed lag

## Priority Decision

- **A Priority (record now):** 4+ flags + lead-dependent business
- **B Priority (record this week):** 2-3 flags + decent fit
- **C Skip for now:** <2 flags or low commercial fit

## Capture These Notes

- Business name + URL
- Top 3 issues
- Why this business likely cares
- Recommended audit angle
