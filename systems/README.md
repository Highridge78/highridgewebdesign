# Highridge Systems Layer

This directory contains operator systems only.
It is intentionally separated from `/app` / production code.

## Structure

- `/audits` - audit checklists, scoring, Loom talk tracks, written summary templates
- `/outreach` - outreach sequence, subject lines, DM variants
- `/sales` - offer framing, qualification, implementation workflow
- `/tracking` - CRM schema and usage SOP
- `/proof` - authority-building strategy without fabricated proof
- `/calls` - discovery and audit-to-proposal conversion system
- `/workflow` - daily, weekly, and end-to-end operating cadence
- `/templates` - reusable snippets for prospect notes and send-ready copy

## Non-negotiables

1. No fabricated proof.
2. No fake urgency.
3. No generic “marketing fluff.”
4. Every Loom and summary must tie recommendations to trust, lead flow, and booked work.
