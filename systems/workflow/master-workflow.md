# Master Workflow (End-to-End)

1. Identify local lead-dependent business
2. Rapid-screen website
3. Score fit + opportunity
4. Record personalized Loom audit
5. Send Loom + written summary
6. Execute follow-up sequence
7. Handle replies and book call
8. Run discovery/audit call
9. Decide fit and propose next step
10. Track outcome + recycle learnings into scripts

## Core System Rule

Every stage must answer: "What is the next action and when?"
