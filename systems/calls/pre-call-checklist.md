# Pre-Call Prep Checklist

- [ ] Rewatch sent Loom (2x speed) and note top 3 issues
- [ ] Review CRM notes and response context
- [ ] Confirm business model and high-value services
- [ ] Prepare 3 discovery questions tied to their goals
- [ ] Prepare 2 scope paths (phased vs full)
- [ ] Open call doc with agenda + note template
- [ ] Confirm next-step CTA before call starts
