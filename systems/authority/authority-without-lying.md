# Honest Authority Framework

## Rules

1. Label concept work as concept work.
2. Never imply client relationship where none exists.
3. Never invent outcomes, numbers, or social proof.
4. Anchor credibility to depth of analysis and clarity of execution.

## Message Pattern

- Observation
- Impact
- Fix direction

## Example

"This homepage likely leaks intent because the offer is unclear above the fold. A clearer promise + service-area cue + primary CTA would increase qualified action probability."

## Trust Through Process

- Publish checklists
- Show repeatable audit method
- Be explicit about assumptions
- Use buyer-language, not design jargon
